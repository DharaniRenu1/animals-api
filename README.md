# Animals API

Animals API is a Spring Boot web application with a Swagger documented API.

# Getting Started

### Prerequisites

First install:

* Java 13
* Maven 3.6.3+

### Commands

Run the service

    mvn spring-boot:run
    
Tests with coverage

    mvn clean test
